Spantail は作業可視化プラットフォームです。人の作業を**作業記録**として、AI
エージェントの活動を**スパン**として記録し、その両方を**レポート**にまとめます。
このガイドでは日々の Web アプリの使い方 — サインイン、作業の記録、レポートの閲覧、
ツール連携 — を扱います。

## サインイン

インスタンスの URL を開き、ログイン画面でサインインします。インスタンスの設定に
応じて、次の方法が使えます。

- **メールアドレスとパスワードでサインイン** — 既定の方法です。
- **Google / GitHub でサインイン** — 管理者がソーシャルログインを有効にしている
  場合。
- **招待を受け入れる** — 招待リンクを受け取ったら、それを開いてアカウントを作成し
  インスタンスに参加します。名前とパスワードを設定するか、Google / GitHub で受け
  入れられます。

:::note[スクリーンショット]
ログイン画面。メールアドレス／パスワードのフォームとソーシャルログインのボタン。
🚧 画像は後日追加します。
:::

パスワードを忘れた場合は、**パスワードをお忘れですか** からリセットリンクを受け
取れます。

## 画面の構成

サインインすると、画面には常に2つの領域があります。

- **サイドバー（左）** — ワークスペース単位。上部に**ワークスペース切替**、その下に
  ワークスペースのナビゲーション、最下部に[設定ハブ](/ja/guides/account-preferences/)
  を開く**設定**の歯車が1つあります。画面が狭いときはアイコンに折りたたまれます。
- **ヘッダー（右上）** — ユーザー単位の領域。**レポート**、**受信箱**、**ユーザー
  メニュー**（アカウント、サインアウト）。

:::note[スクリーンショット]
アプリ全体。左のワークスペースサイドバーと、右上にレポート・受信箱・ユーザー
メニューを持つヘッダー。
🚧 画像は後日追加します。
:::

## 主要な概念の早見表

| 概念 | 内容 |
|---|---|
| **ワークスペース** | 組織の単位（チーム・部門・顧客など）。複数に所属できます。 |
| **プロジェクト** | ワークスペースの下位区分。作業記録はプロジェクトに紐付けるか、ワークスペース全体のままにできます。 |
| **作業記録** | 人の作業の記録。日付・所要時間・説明・任意のメモ・タグを持ちます。 |
| **エージェント活動** | スパンとして取り込んだ AI エージェントのセッション。人の作業と並んでタイムラインに表示されます。 |
| **レポート** | テンプレート・フィルタ・期間から作る不変のスナップショット。閲覧・共有・議論ができます。 |

## 最初の作業を記録する

1. サイドバーからワークスペースのダッシュボードを開きます。
2. **+** ボタンをクリック（または <kbd>C</kbd> キー）します。
3. 所要時間と短い説明を入力し、必要ならプロジェクトを選んで保存します。

これだけで、記録がタイムラインに表示されます。詳しくは
[作業を記録する](/ja/guides/logging-work/) を参照してください。
