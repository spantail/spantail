専用の **Claude Code プラグイン**を予定しています。Claude Code のセッションをエージェント
活動として自動で取り込み、インスタンスに接続する以外の手動設定を不要にする想定です。

それまでの間も、MCP 経由で Claude Code（や他の AI エージェント）の活動を取り込めます。

- `claude mcp add spantail -- spantail mcp` で Claude Code を接続します — [MCP](/ja/guides/tools/mcp/)
  を参照してください。
- エージェントを登録してトークンを取得します — [エージェント記録](/ja/guides/capturing-agents/)
  を参照してください。
