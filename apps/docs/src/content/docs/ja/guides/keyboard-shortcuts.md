---
title: キーボードショートカット
description: Web UI を操作・移動するためのキーボードショートカット。
---

Web UI には、よく使う操作のための単一キーのショートカットがいくつかあります。テキスト
入力欄に入力して**おらず**、ダイアログやメニューが**開いていない**ときに動作します。

| キー | 動作 |
|---|---|
| <kbd>C</kbd> | 現在の画面に応じて新規作成します — ダッシュボードでは作業エントリ、レポート画面ではレポート。 |
| <kbd>J</kbd> | リストの選択を**下**へ移動します。 |
| <kbd>K</kbd> | リストの選択を**上**へ移動します。 |
| <kbd>O</kbd> | ハイライトされた項目を開きます。 |
| <kbd>X</kbd> | ハイライトされた行のチェックを切り替えます（一括選択に対応したリスト）。 |
| <kbd>S</kbd> | 開いているメッセージのスターを切り替えます（メッセージ）。 |
| <kbd>E</kbd> | 開いているメッセージをアーカイブ、または受信トレイに戻します。 |
| <kbd>D</kbd> | 開いているメッセージをゴミ箱へ移動（または元に戻す）します（メッセージ）。エージェントのセッション一覧では、チェックしたセッションを削除します（先に確認ダイアログが開きます）。 |

リスト用のショートカット（<kbd>J</kbd> / <kbd>K</kbd> / <kbd>O</kbd>）は、タイムライン・
レポート一覧・受信トレイ・エージェントのセッション一覧など、選択に対応したリストで使えます。

メッセージでは、<kbd>S</kbd> / <kbd>E</kbd> / <kbd>D</kbd> が開いているメッセージに作用します
（スター・アーカイブ・ゴミ箱へ移動）。いずれも切り替えなので、スター付き・アーカイブ・ゴミ箱の
各フォルダでは同じキーで元に戻せます。アーカイブまたはゴミ箱へ移動すると選択が次のメッセージへ
進んで開くので、キーボードだけで連続して片付けられます。フォルダの末尾では閲覧ペインが閉じます。

詳細パネルを開いている間は — タイムラインの作業エントリでも、エージェント活動ページの
セッションでも — <kbd>↑</kbd> / <kbd>↓</kbd> でリスト内をそのまま移動でき、パネルの
表示も追従します。

エージェントのセッション一覧では、<kbd>X</kbd> で自分のセッションを複数選択できます。
セッションをチェックした状態では、<kbd>C</kbd> が選択内容からプレフィルされた作業エントリ
ダイアログを開き（作成された作業エントリはそのセッション群に関連付けられます）、<kbd>D</kbd> が
削除の確認を開きます。選択にプロジェクトが混在している場合や上限を超えている場合は、
対応するボタンと同様にどちらも動作しません。
