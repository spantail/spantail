---
title: キーボードショートカット
description: Web UI を操作・移動するためのキーボードショートカット。
---

Web UI には、よく使う操作のための単一キーのショートカットがいくつかあります。テキスト
入力欄に入力して**おらず**、ダイアログやメニューが**開いていない**ときに動作します。

| キー | 動作 |
|---|---|
| <kbd>C</kbd> | 現在の画面に応じて新規作成します — ダッシュボードでは作業記録、レポート画面ではレポート。 |
| <kbd>J</kbd> | リストの選択を**下**へ移動します。 |
| <kbd>K</kbd> | リストの選択を**上**へ移動します。 |
| <kbd>O</kbd> | ハイライトされた項目を開きます。 |

リスト用のショートカット（<kbd>J</kbd> / <kbd>K</kbd> / <kbd>O</kbd>）は、タイムライン・
レポート一覧・受信箱・エージェントのセッション一覧など、選択に対応したリストで使えます。
