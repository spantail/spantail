---
title: Keyboard shortcuts
description: Keyboard shortcuts for navigating and acting in the web UI.
---

The web UI has a few single-key shortcuts for common actions. They work when
you're **not** typing in a text field and **no** dialog or menu is open.

| Key | Action |
|---|---|
| <kbd>C</kbd> | Create a new item for the current screen — a work entry on the dashboard, a report on the reports screen. |
| <kbd>J</kbd> | Move the selection **down** in a list. |
| <kbd>K</kbd> | Move the selection **up** in a list. |
| <kbd>O</kbd> | Open the highlighted item. |

The list shortcuts (<kbd>J</kbd> / <kbd>K</kbd> / <kbd>O</kbd>) apply to the
lists that support selection, such as the timeline, the reports list, your
inbox, and an agent's sessions list.
